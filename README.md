# Ardent Pack Builder

A powerful, web-based tool for creating custom RPG pack pages. Build, customize, and generate code for your RPG world with ease.

## Features

- **Live Preview**: See your changes in real-time as you build.
- **Customizable Stats**: Add and configure stats like Alignment, Territories, and Leaders.
- **Dynamic Roster**: Build a full pack hierarchy with names, ranks, and descriptions.
- **Styling Options**: Customize colors, fonts, background images, and textures.
- **Mobile Friendly**: The generated code is designed to look great on all devices.
- **One-Click Export**: Copy the generated HTML/CSS and paste it into your RPG site.

## Installation

Simply clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/ardently/ardentpackbuilder.git
cd ardentpackbuilder
# Open index.html in your browser
```

## Usage

1. Fill out the **Header Info** with your pack's name, slogan, and banner.
2. Configure your **Stats Bar** to highlight key data points.
3. Add your pack's **Lore & Info** (History, Rules, etc.).
4. Add **Sidebar Items** for quick links and news.
5. Define your **Pack Ranks** and add members to your roster.
6. Customize the look and feel in the **Styling** section.
7. Click **Copy Code** and you're ready to go!
